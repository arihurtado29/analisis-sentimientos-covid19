#  Análisis de Sentiminetos sobre el Impacto del COVID-19
Proyecto de análisis exploratorio de datos en el que analicé las respuestas obtenidas de una encuesta utilizando escalas de Likert y visualizaciones para identificar tendencias y comprender la percepción de los participantes sobre el impacto del COVID-19.

## Objetivo
Analizar las respuestas obtenidas en una encuesta sobre el impacto del COVID-19 para identificar patrones y tendencias mediante técnicas de análisis exploratorio de datos. A partir de la información recopilada, se buscó interpretar la percepción de los participantes utilizando escalas de Likert y representaciones gráficas que facilitaron el análisis de los resultados.

## Dataset
El dataset utilizado "covid_impacto_sentimientos-200_respuestas", el tipo de dato es una encuesta, contiene un registro de 200 respuestas y sus variables son información demográfica y preguntas relacionadas con el impacto del COVID-19 evaluados mediante escalas de Likert.
